# Practico 3 - Programacion IV

### Comandos para construir el proyecto

1. Clonar el repositorio
```bash
git clone https://github.com/Mada98/Programacion-IV.git
```

2. Entrar a la carpeta del Practico 3
```bash
cd /Practico 3/cafeteria/
```

3. Instalar las dependecias:
```bash
npm i
```

### Comando para correr los test

+ Test
```bash
npm run test
```
